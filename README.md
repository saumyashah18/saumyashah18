## 🧠 What I'm Building Right Now

At **CLEF (Centre for Learning Futures), Ahmedabad University**, I'm co-developing **AIssociate** — an in-house AI assessment platform built on a custom **GraphRAG pipeline**, FAISS vector indexing, and Gemini Pro.

It replaces expensive SaaS EdTech tools, integrates natively with the university's LMS and Safe Exam Browser, and actually adapts to how a student is thinking — not just what they answer. Already piloted in a live mid-semester assessment with real teachers and students at AU.

### 📊 PerformIQ
> Workforce analytics — because "productivity" deserves a real formula.

`React` `Tailwind CSS` `Node.js` `Express.js` `SQLite` `Python` `pandas` `sklearn` `scipy` `Firebase Auth`
